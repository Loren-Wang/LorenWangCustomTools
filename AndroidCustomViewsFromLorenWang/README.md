android 自定义框架

<h3>1、CircleImageView---(圆形图片控件)</h3>
ps：仅显示圆形图片以及边框，通过布局文件控制，不支持使用代码控制图片边框

<h3>2、RoundedRectangleImageView---(圆角图片控件)</h3>
ps：不支持使用代码控制图片边框以及各个角度，如果定义了通用角度则其他的单角设置失效，如果没有定义通用角度则单角定义启用
